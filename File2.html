Introduction

This Hackathon is designed to familiarize participants with two powerful CAST tools: CAST Highlight and CAST Imaging. The event is structured in two parts:

Hack-1: CAST Highlight - Portfolio Analysis: This section focuses on portfolio analysis of applications. Participants will learn how to:

  • Conduct comprehensive application assessments
  • Generate modernization insights
  • Receive targeted recommendations
  • Identify suitable AWS services for migration and optimization

Hack-2: CAST Imaging  - Application Analysis for Modernization: In this part, participants will explore an interactive tool for in-depth application analysis. They will learn to:

  • Visualize and understand application architecture
  • Identify key components and their interactions
  • Navigate complex codebases efficiently
  • Gain insights into potential areas for improvement or refactoring

By engaging with both tools, participants will develop a holistic approach to application modernization, combining high-level portfolio management with detailed architectural analysis. This hands-on experience will equip them with valuable skills for optimizing and transforming their software ecosystems.

Prerequisite

1. AMI deployed in all Isengard accounts - 1 designated account for each team 
2. CAST Highlight - username and password is available per group.
3. RDP Client to be installed on your laptop for remote access - for Mac user 


Important notes: The entire hackathon hands-on process needs to be done inside the Windows EC2 instance pre-deployed inside the Isengard account.

Hack-1 : CAST Highlight - Portfolio analysis

Introduction

In this session, participants will explore the capabilities of CAST Highlight, a powerful tool designed to streamline and enhance cloud migration strategies. The session will demonstrate how CAST Highlight can rapidly assess the cloud readiness of large application portfolios, regardless of their current hosting environment or technology stack. Attendees will learn to leverage this tool to analyze hundreds of custom applications within a week, providing accurate insights for both on-premise systems and those already migrated to major cloud providers like AWS, Google, or Azure.
The session will also cover how CAST Highlight can be used beyond initial assessment, serving as a continuous monitoring solution throughout the entire cloud migration journey. Participants will discover how to track the progress of their cloud transformation programs, ensuring ongoing optimization. Additionally, the session will explore CAST Highlight's features for continuous analysis of open-source safety and overall software health across the entire application portfolio. This comprehensive approach enables organizations to maintain visibility and control as their applications evolve and modernize in the cloud environment, ensuring long-term success and security in their digital transformation efforts.

Section 6 : Application Analysis Results

Analysis results can be viewed from different perspective via different menus provided. 

SOFTWARE HEALTH

Software Health indicates how your application complies with programming best practices that increase resiliency, improve agility and reduce complexity. Software Health score is the straight average of Software Resiliency, Software Agility and Software Elegance scores.

  • Software Resiliency:

Software Resiliency indicates programming best practices that make software bullet-proof, more robust and secure. This index is derived from technology-specific code analysis which searches for the presence of code patterns and bad programming practices which may compromise the reliability of the software in the short term. Higher the Software Resiliency, lower is the likelihood of defects to occur in production.

  • Software Agility:

Software Agility indicates the easiness of a development team to understand and maintain an application. This index is derived from technology-specific code analysis which searches for the presence of embedded documentation and code readability good practices.

  • Software Elegance:

Software Elegance measures the ability to deliver software value with less code complexity. A low Software Elegance score indicates decreased quality in the code resulting in higher defects which may become costly to fix in the mid-term.

CLOUD MATURITY

CAST Highlight helps automatically segment your application portfolio into Cloud migration buckets, within a manner of seconds. The Portfolio Advisor for Cloud dashboard is an automated and smart way to segment an application portfolio for Cloud migration. 

The five Portfolio Advisor for Cloud segments are defined as follows:

Rehost: A recommendation to change the infrastructure configuration of the application in order to “Lift & Shift” it to the cloud using Infrastructure as a Service (IaaS).
Refactor: A recommendation to perform modest modifications of the application code without changing the architecture or functionality so that it can be migrated to the cloud in a container using Container as a Service (CaaS) or using Platform as a Service (PaaS).
Rearchitect: A recommendation to dramatically modify the application code thereby altering the architecture to improve the health of the application and enable it to be migrated to the cloud using Platform as a Service (PaaS) or deployed serverless using Function as a Service (FaaS).
Rebuild: A recommendation to discard the code of the application and develop it again in the cloud using Platform as a Service (PaaS) or serverless using Function as a Service (FaaS).
Retire: A recommendation to discard the application altogether or potentially replace it with a commercial Software as a Service (SaaS) alternative.

The Cloud Migration Wave Advisor combines both strategical and tactical insights gathered for each application analyzed by CAST Highlight to automatically propose an optimized sequence, in four different waves:

  • Wave 1 – Quick Wins: Migrate now with minimal effort and risk
  • Wave 2 – Core Cloud: Migrate next while also modernizing
  • Wave 3 – Long Term Bets: Migrate and then modernize later
  • Wave 4 – Pursue Later: Modernize first and then migrate 

Cloud Service Recommendations: CAST has leveraged Software Intelligence insights, in partnership with Cloud experts from Amazon Web Services, in a series of workshops to translate them into specific recommendations for adopting various cloud services on AWS.

Hack-2: Cast Imaging - Application Analysis for Modernization

Introduction

CAST Imaging is a sophisticated, interactive tool designed to reverse-engineer and visualize complex software systems. It creates detailed, interactive architecture blueprints that allow users to navigate through intricate application structures, from high-level services down to object-level granularity. The tool automatically analyzes and maps out the technology stack, application layers, and interdependencies between components, providing a comprehensive view of the entire system. One of its key features is the ability to generate and display end-to-end transaction flows, enabling users to trace processes from the user interface through various intermediate layers to the database. Acting as a robust technical knowledge base, CAST Imaging offers powerful search capabilities, documentation features for user annotations, tagging functionalities to categorize technical objects, and aggregation tools for creating functional views or designing future architectures. It also provides insights into inter-application relationships, such as API calls and shared data resources. This comprehensive approach makes CAST Imaging an invaluable asset for software architects, developers, and decision-makers seeking to understand, analyze, and optimize complex application ecosystems.
